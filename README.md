## Contributing

To run this app on your local machine, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/zTomz/supaquiz.git
   ```

2. **Navigate to the cloned repository:**

   ```bash
   cd supaquiz
   ```

3. **Install dependencies:**

   ```bash
   flutter pub get
   ```

4. **Run the app:**

   ```bash
   flutter run
   ```
